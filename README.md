The S3Main is the first officially supported [OpenRemise](https://openremise.at) board.

### Features
- WiFi-capable ESP32-S3 SoC
- 10-22V DC supply
- Reverse polarity protection
- Single track output
- Adjustable current limits 0.5, 1.3, 2.7 and 4.1A
- [SUSI](https://normen.railcommunity.de/RCN-600.pdf) interface
- USB-C interface
- RailCom detector
